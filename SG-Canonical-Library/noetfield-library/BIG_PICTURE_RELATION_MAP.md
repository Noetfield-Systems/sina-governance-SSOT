1. **Truth flows down, proposals flow up.** SSOT Index → Canon → SSOT → loops → kernel/line-engine → execution. Failures/learnings flow UP as *proposals only* (learning-proposal, council) — never direct edits. (bidirectional-nervous-system)

2. **The brain is empty + your definitions.** Base model has no competing definitions (advantage) → SSOT v6 becomes its first-class language → locked-definitions is the meaning source → decision-core decides deterministically → Llama (soup) only phrases, walled + sanitized. (deterministic-brain, soup/raw, base-model-first-class-language)

3. **Contracts run, LLMs propose.** Execution Contract holds authority; LLM emits only the Plan Contract schema; Control gate + Blueprint Registry enforce; Executor runs the exact call. (contracts-run-the-system, execution-contract-as-brain)

4. **The floor is immutable.** Verifier/gate/kill-switch/mutation-guard/LOCKED-SSOT can't be edited from inside the loop → self-improvement is safe → 24/7 autonomy survivable. (immutable-floor, targets-vs-blockers)

5. **Everything is proven by receipts, mechanically, by rejection.** Not diagrams (receipts-not-diagrams), not prose (mechanical-not-prose), proven by a failing input the gate rejects (negative-proof), by an independent checker (author≠subject). This provability IS the product (anti-theater → pattern-factory).

6. **Founder deletes work, doesn't do it.** ROC: every manual action = assume system failed → convert to Rule/Pattern/Automation/Receipt. Canon §1 enforces; the whole library exists to make the founder unnecessary to operation. (return-on-cognition, founder-intent-filter)

7. **Run 24/7 pursuing targets; stop only for the floor.** Only genuine harm/impossible = blocker. SUPABASE_URL resolved (see `P2-SSOT/OPEN_BLOCKERS.md`). Everything else = target the running system chases. (targets-vs-blockers)
