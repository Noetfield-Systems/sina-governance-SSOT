# LIVE_SITE_CHANGE_PLAYBOOK_v2.0

## For every live site edit

1. Run CHESS_PASS.
2. Create protected feature ledger.
3. Separate REMOVE ONLY from PRESERVE.
4. Implement patched move.
5. Deploy.
6. Verify live board.
7. Write receipt.
8. Stop.

## Never
- assume repo = live
- assume receipt = truth
- remove features under “clean/minimal”
- declare PASS without live verification
