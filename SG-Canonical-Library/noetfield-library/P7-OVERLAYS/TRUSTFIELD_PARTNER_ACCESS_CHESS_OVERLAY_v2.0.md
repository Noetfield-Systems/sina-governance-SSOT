## 3. Buyer CTA rule

Buyer/product CTAs may be reduced only if Partner Access controls remain stronger and visible.

Allowed:

```text
- reduce Start Sandbox prominence
- remove product closing block from Partner Access
- separate buyer funnel from partner funnel
```

Not allowed:

```text
- remove Sign in
- remove Create account
- remove Partner Access Platform
- remove View application status
- remove Request Partner Access
- remove apply routes
```

