# PHASE_LOOP_BUILD_PLAN_v0.1

**Goal:** a self-improvement loop where a sandbox proposes a change to SourceA's
live Brain, the independent verifier checks it, and only PASS reaches the live Worker.

**First artifact (locked):** the Brain Worker **knowledge bundle**
(`cloud/workers/sourcea-brain-chat-v1/src/knowledge-bundle.json`).
Data, not code — safe to verify by schema + hash. Generated by
`scripts/brain_chatbot_refresh_v1.sh` from `data/CHATBOT_KNOWLEDGE_MANIFEST.json`.

**Live target:** the Cloudflare Worker `sourcea-brain-chat-v1`.

**Verifier (already built):** GitHub-App + secondary-Cloudflare-account Worker,
`sina-governance-ssot-advisory.kazemnezhadsina144.workers.dev`, account
`b7282b4a5c17b84d62e3ef8866b878f8`. Emits receipts; PASS only when receipt fields
prove independence (author≠subject, separate runtime/account, evidence present).

## How Cursor uses this file
Work top-down. For each step: do the SCOPE, prove the DONE-CONDITION with evidence,
report, then propose the next step and wait for one-word go. Do not advance past a
step whose done-condition is unproven. **STEP 10 is founder-gated — propose only.**
Short prompts. Build, don't pad with "do not" lists.
